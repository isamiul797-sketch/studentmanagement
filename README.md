# Student Management (CRUD Project)

A simple Django project to manage students with basic CRUD operations.

## Features
- Add new student
- View all students
- Update student details
- Delete student

## Tech Stack
- Django (Python)
- SQLite (default database)
- HTML, CSS (basic templates)
